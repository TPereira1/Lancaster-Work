Problem 1: Working with Arrays of Integers

You’ll need to create a program that allows the user to enter 10 numbers into an integer array.

Then extend your program to do the following:

  1. print out all of the numbers in the array
  2. print the smallest number in the array
  3. print the largest number in the array
  4. print the position in the array of the smallest and largest numbers
  5. calculate the arithmetic mean (recall, the arithmetic mean is the sum of the numbers divided by ‘n’, the number of numbers)

General approach. Think about the overall design of each part of the task. Solve each one in turn and ensure your code works as desired before moving onto the next part. As ever, we highly recommend sketching out each algorithm on paper first.

Hacker edition

‘Bubble sort’ is a well-known sort algorithm that is used to put a list of items into ascending order by ‘bubbling’ the smallest item to the top (first array position), then the next smallest to the next slot (second position) etc. It stops when no swaps are made (all items sorted).

Pseudocode is as follows (see animation):

for i = 1:n,
    swapped = false
    for j = n:i+1,
        if a[j] < a[j-1],
            swap a[j,j-1]
            swapped = true
    → invariant: a[1..i] in final position
    exit if not swapped
end
  1. Write a C implementation of bubble sort using arrays. You should generate random numbers to populate your array, and output the sorted list when done.
  2. Count the number of times through the loop and the number of swaps made, and output this at the end of the program.
  3. Experienced programmers only recommended. Write an ‘ordered insert’ that ensures that elements are only inserted in the correct position in the array (maintaining ascending order), so the sort is no longer necessary.
  4. (optional) Generate random lists and time the operation of your first approach (1) vs. your second from (3) above. Which has better performance timed over 10 iterations? The gettimeofday function can be used to do millisecond or better time measurements.
